# Electromagnetic Engineering
UESTC大学物理期末自拟编程题
——在电流强度变化的直导线产生的磁场中任意形状线框运动产生电动势

Visual&Calculus文件夹
实现数值计算以及可视化的程序——Physical_calculation.py
三张图片即由该程序生成的可视化图像；
另外有两个文件是将图像文件打包作为UI资源

GUI文件夹
MainWin.py和MainWin.ui是UI的界面文件、Project.py是UI的逻辑文件
Changemain.py用于界面调参，
可执行文件由于和DDL动态链接挂钩
放在了GUI\Project\Project.exe，做了图标以示区分
